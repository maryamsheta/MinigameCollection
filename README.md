This Minigame Collection is a growing compilation of simple yet engaging browser-based games built with p5.js.
#### **Current Games**  
- **Match The Ring** – A precision-based game where players attempt to match the size of a ring.  
- **Place All Squares** – A puzzle game requiring players to correctly align rotating squares.  
- **Hot or Cold** – A search-based game where players must locate a hidden object using color-coded hints.
  
**Note: Some interactions may not work properly on touch devices.**
